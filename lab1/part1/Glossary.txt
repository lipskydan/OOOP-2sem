                                   Glossary
__________________________________________________________________________

User - користувач, який увійшов у систему.
User with Business Account - користувач, який має можливість слідкувати за популярністю свого акаунта.
Guest - людина, яка можливо стане користувачем. 
__________________________________________________________________________

class User Profile - головний клас, який відповідає за інформативну частину сторінки користувача та весь інший функціонал.
class User - клас, який зберігає основну (базову) інформацію про користувача (User).
class User  with Business Account - клас, який надає User додаткові властивості.
classes Follower & Following - класи, які зберігать інформацію про користувачів, які слідкують за User, або інформацію користувачів, які є цікавими для User.
class News - клас, який відповідає за показ стрічки новин.  
class Account’s ID - клас, який відповідає за пошук конкретного користувача.
class Content - клас, який відповідає за додавання нового відео та/або фото на сторінку User.
classes Photo & Video - класи, які використовує class Content, для отримання самих матеріалів.
class Comment - клас, який відповідає за створення нових та видалення наявних коментарів користувача.
class Like -  клас, який відповідає за відображення вподобання користувача.

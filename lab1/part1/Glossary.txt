                                   Glossary
__________________________________________________________________________

User - користувач, який увійшов у систему.
User with Business Account - користувач, який має можливість слідкувати за популярністю свого акаунта.
Guest - людина, яка можливо стане користувачем. 
__________________________________________________________________________

User Profile - відповідає за інформативну частину сторінки користувача та весь інший функціонал.
User - зберігає основну (базову) інформацію про користувача (User).
User  with Business Account - клас, який надає User додаткові властивості:
	* перевірка статистики свого аккаунта (відвідуваняб популярність, тощою
	* рекламування когось або чогось
	* просування постів
Follower & Following - зберігать інформацію про користувачів, які слідкують за User, або інформацію користувачів, які є цікавими для User.
News -відповідає за показ стрічки новин.  
Account’s ID - відповідає за пошук конкретного користувача.
Content - відповідає за додавання нового відео та/або фото на сторінку User.
Photo & Video - використовують class Content, для отримання самих матеріалів.
Comment - клас, який відповідає за створення нових та видалення наявних коментарів користувача.
Like - відповідає за відображення вподобання користувача.
